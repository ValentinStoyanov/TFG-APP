Welcome

This app is developed for the purpose of connecting and displaying data from a back end which can execute reports from a instance of Neo4j.

To start executing reports navigate to "Reports" and select the desired one.

You can navigate to "Historics" to see the history of executed reports, see the content of the selected report or delete it.

Neo4J + Spring Boot Connector + Framework

This app uses Sprig Data which provides convenient access to Neo4j including object mapping, Spring Data repositories, conversion, transaction handling, reactive support, and more.
